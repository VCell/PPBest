# PPBest - 魔兽世界宠物对战一键助手

适用于国服熊猫人版本。主要用于pvp对战场景，方便成就党刷胜场，和利用小宠物pvp升级。

默认按键是F8,可在设置-插件-PPBest中修改

## 场景一：两人互刷低等级宠物对战

适用于用小宠物pvp升级，或成就党速刷pvp5000胜的成就

**使用方法：**
1. 双方玩家都需要安装本插件
2. 队伍配置要求：
   - 使用1-2个满级宠物带队，满级宠物放在前面
   - 双方队伍的总等级相同
   - 需要吃经验或胜场的一方将某个宠物名字改为"我要经验"或"我要胜场"
   - 辅助的一方将某个宠物名字改为"我要送"
3. 双方都不停按排队宏和F8（或修改后的快捷键）即可

**说明：**
1. 人物有经验的标准是，战斗时长超过一分钟。因此“我要经验”的情况下，辅助一方会在选人后等一分钟认输。
2. 如果在选人阶段对手认输，所有宠物会均分经验。如果进入到战斗阶段，上场宠物就会有经验。为了避免低等级宠物升级导致匹配不顺畅，双方都要将低等级宠物后置。辅助方的插件会在选人后认输
3. 宠物pvp对战的匹配是有按胜场数的隔离的。如果两人无法匹配上，胜场低的一方先去单刷，提升到100胜以后再尝试（具体数值不能肯定）


## 场景二：单人刷满级对战

用于刷全25宠物的1000胜成就，或没人互刷的情况下升级和刷5000胜。装上插件配好阵容，排队和F8一直按即可。

**战斗逻辑：**
1. 3只PP节点雏龙，有相对完整的策略，会考虑天气，对方血量，属性等。策略复杂不详细说明
2. 下面推荐表格中的宠物以任意组合组队，会按照队伍次序上场，技能按固定优先级释放。
3. 表格外的宠物，或表格内宠物没带指定技能的，会随机释放技能
