# PPBest - 魔兽世界pvp宠物对战一键助手

适用于国服熊猫人版本。主要用于pvp对战场景,一键排队、进场、选人、战斗，方便成就党刷胜场，和利用小宠物pvp升级。

默认按键是F8,可在设置-插件-PPBest中修改

## 场景一：两人互刷低等级宠物pvp对战

可用于给大量宠物升级、给玩家升级、刷宠物pvp胜场数。需要有一名玩家作为辅助方。双方都安装本插件，辅助方在选项-插件-PPBest中设置模式为“辅助方”并在“辅助目标ID”处填写主刷的玩家id-服务器，主刷方按照自己的需求选择模式。

**互刷模式：**
1. 我要胜场：用于速刷5000胜成就、好运符、战斗石。辅助方进场就会认输。主刷方阵容等级：25、25、任意<23级。注意3号位如果高于23级会被插件自动换掉，防止匹配到别人被升级
2. 我要角色经验：用小宠物pvp升级。辅助方会进场后等一分钟认输（人物有经验的标准是，战斗时长超过一分钟）。主刷方阵容等级：25、25、任意<23级。
3. 我要宠物等级: 用于批量给小宠物升级。辅助方进场后认输。主刷方需要提前把所有要升级的小宠物设置为偏好，插件会自动把偏好列表中不满级的宠物换到阵容的三号位置。主刷方阵容等级：25、20、要升级的小宠物。

**说明：**
1. 辅助方会在每局开始前密语询问辅助目标(即主刷方)的阵容等级，然后按对应等级配置阵容，故辅助方需要提前准备好所有等级的小宠物至少1个。
2. 自动换宠物的功能和rematch插件冲突。在互刷过程中，需要关闭rematch插件并清空宠物手册的左上角筛选条件。否则可能无法自动换宠
3. 宠物pvp对战的匹配是有按胜场数的隔离的。如果两人无法匹配上，胜场低的一方先去单刷，提升到100胜以后再尝试（具体数值不能肯定）


## 场景二：单人刷满级对战

用于刷全25宠物的1000胜成就，或没人互刷的情况下升级和刷5000胜。装上插件配好阵容，F8一直按即可。
有“单刷-AI”和“单刷-基础策略”两种模式。在宠物和电脑配置支持的情况下，推荐使用“单刷-AI”模式，胜率明显高。

基础策略会按照预设优先级释放技能，不认识的技能会随机释放；AI默认会按照算法释放技能，宠物对战过程中电脑会有卡顿。
